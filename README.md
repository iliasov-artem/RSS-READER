## RSS-Reader - приложение для чтения rss новостей, позволяет добавлять несколько каналов, а так же автоматически следит за обновление ленты.

## Навыки и технологии

+ Браузерная технология (webpack)
+ Bootstrap
+ AJAX (axios)
+ События, работа с DOM
+ Валидация
+ Деплой (surge.sh)
